Student Management System
A simple full-stack web application to manage student records efficiently. Built using the MERN stack (MongoDB, Express, React, Node.js) with full CRUD functionality.
🚀 Features
➕ Add new students
📋 View all student records
🔍 Search students
✏️ Update student details
❌ Delete student records
⚡ REST API integration between frontend and backend
🛠️ Tech Stack
Frontend:
React.js
Axios
HTML, CSS, JavaScript
Backend:
Node.js
Express.js
Database:
MongoDB
Mongoose
